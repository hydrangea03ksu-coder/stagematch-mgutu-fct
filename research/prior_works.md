# Научный задел МГУТУ по теме StageMatch

## Релевантные выпускные квалификационные работы (ВКР)

### 1. ВКР «Моделирование мягких датчиков кристаллизации тростникового сахара с использованием двойной регрессии опорных векторов»
- **Автор:** Исакин А.А.
- **Год:** 2025
- **Руководитель:** Петров С.М.
- **Кафедра:** Автоматизации и IT
- **Ключевые методы:** SVR (Регрессия опорных векторов), машинное обучение, анализ данных
- **Результаты:** Модель прогнозирования параметров производства
- **Применимость к StageMatch:** Метод SVR будет адаптирован для прогноза успешности выступления артиста на основе исторических данных (посещаемость, жанр, цена билетов).

### 2. ВКР «Разработка информационной системы сбалансированной карты стратегических показателей компании»
- **Автор:** Белякова А.И.
- **Год:** 2025
- **Руководитель:** Акимова И.В.
- **Кафедра:** Корпоративных информационных систем
- **Ключевые методы:** BPMN, UML, агрегация KPI
- **Результаты:** Прототип системы для стратегического управления
- **Применимость к StageMatch:** Методология проектирования корпоративных B2B-систем будет использована при создании архитектуры платформы.

### 3. ВКР «Прогнозирование продаж товаров в компании с помощью Deductor»
- **Автор:** Неизвестно
- **Год:** 2023
- **Кафедра:** Бизнес-аналитики
- **Ключевые методы:** Data Mining, прогнозные модели
- **Результаты:** Модель прогноза спроса для аптечной сети
- **Применимость к StageMatch:** Общий подход к прогнозной аналитике для бизнес-задач.

## Выводы по научному заделу
1. **Прямых работ** по теме автоматизации подбора артистов не обнаружено.
2. **Смежный задел** присутствует в областях:
   - Машинное обучение для прогнозирования (ВКР Исакина А.А.)
   - Разработка корпоративных B2B-систем (ВКР Беляковой А.И.)
   - Прогнозная аналитика (ВКР по прогнозированию продаж)
3. **Рекомендуется** объединить методы из этих работ:
   - Взять за основу **метод SVR** из ВКР Исакина А.А.
   - Использовать **методологию проектирования** из ВКР Беляковой А.И.
